# Installation Instructions
you will more than likely need an API key from thegamesDB.net

**To install the program type the following into your terminal:**
 git clone https://github.com/Wyatt-Cox-student/SDEV265-Project.git

**In order to make everything work you might need to run these installation commands in your terminal:**
pip install tk
pip install customtkinter
pip install requests 
pip install pillow
